# KMT2A-MLLT3 AML OS/RFS Risk Stratification Tool

This is the sanitized static web calculator package for GitHub Pages.

## Contents

- `index.html`
- `app.js`
- `styles.css`
- `model_artifacts/model_bundle.js`
- `logo.png`
- `.nojekyll`

The package does not include source CSV files, scripts, output folders, or `model_bundle.json`.

## Privacy Notes

- Real `reference_cases` have been removed from the published model bundle.
- The "Use example" button uses a hand-written synthetic clinical example, not a study patient.
- The model parameters remain visible to visitors because this is a pure front-end static calculator.

## GitHub Pages

Upload the contents of this folder to a public GitHub repository root, then enable GitHub Pages from the repository root branch.
